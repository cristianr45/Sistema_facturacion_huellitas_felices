# Sistema de Facturacion - Huellitas Felices

Descripcion:
Este es un proyecto de integracion de modulos para una veterinaria. El sistema permite gestionar el catalogo de servicios, registrar clientes con sus mascotas y generar comprobantes de pago en. Todo esta conectado a una base de datos MySQL mediante un servidor en Node.js.

Tecnologias:
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js y Express
- Base de Datos: MySQL


Instalacion y Uso:

1. Base de Datos:
Importar el archivo .sql adjunto en MySQL Workbench. El esquema debe llamarse huellitasfelices.

2. Dependencias:
Ejecutar el comando "npm install" en la terminal dentro de la carpeta del proyecto para instalar express y los controladores necesarios.

3. Ejecucion:
Iniciar el servidor con el comando "node server.js". Una vez encendido, abrir el archivo login.html en el navegador para ingresar al sistema.
